# 夏夢 (Natsuyume)

A modern offline music player built with Flutter and C++, inspired by Material 3 Expressive and designed around the idea that local music deserves a first-class experience.

Natsuyume combines a highly customizable user interface, advanced library management, synchronized lyrics support, multiple queues, dynamic theming, and a native playback core into a lightweight music player focused on personal collections rather than streaming services.

> "May God's will be done."

---

## Features

### Library Management

* Automatic folder scanning
* Album, Artist, Playlist, and Queue organization
* Metadata editing
* Artist and playlist profile customization
* Multi-select operations

### Playback

* Gapless playback
* Multiple queues
* Queue reordering
* Mini player
* Full-screen Now Playing interface
* Favorites support
* Playback statistics

### Lyrics

* Synchronized LRC lyrics support
* Embedded lyrics support
* Lyrics editor
* Timestamp editing

### Customization

* Material 3 Expressive design
* Dynamic color support
* Multiple theme styles
* Secret unlockable themes
* Manual theme customization

### Data Management

* User data persistence
* Backup and restore support
* Playlist import/export
* CSV import/export
